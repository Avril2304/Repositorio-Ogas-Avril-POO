# Ejercicio 01 - Planificador de Trabajos Prácticos

Aplicación desarrollada en Qt (C++) que permite gestionar trabajos prácticos con persistencia local.

## Funcionalidades
- Login de usuario
- Gestión de tareas (CRUD)
- Filtros por estado y prioridad
- Persistencia en archivos CSV y JSON
- Manejo de sesiones
- Historial de acciones

## Estructura
- codigo/: lógica de la aplicación
- datos/: almacenamiento local

## Tecnologías
- C++
- Qt (QWidget)
- JSON / CSV
